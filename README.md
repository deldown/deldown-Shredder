# Installation
[(Back to top)](#table-of-contents)

Wie man sich den Source Code von deldown Shredder von Github CLI cloned:

```shell
gh repo clone deldown/deldown-Shredder
```

Auf Git:

```shell
git clone https://github.com/deldown/deldown-Shredder/
```

Oder auf Windows kann man den Installer öffnen, und dann kann man die exe Datei von deldown Shredder ausführen.
